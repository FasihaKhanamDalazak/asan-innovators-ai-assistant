# Asan Innovators — Case Study: Megaa App

Asan Innovators is a technology company headquartered in Nizampet, Hyderabad, Telangana, India. This is one of our 15 portfolio case studies.

## Project Identity

- **Client:** Megaa
- **Client location:** Hyderabad
- **Year:** 2024
- **Category:** Web & Mobile Apps

## Project Description

We built Megaa, a service booking mobile application connecting customers with local service providers in Hyderabad. The app features seamless booking flows and real-time availability.

## Highlighted Results

- **iOS and Android platform coverage**
- **Hyderabad market**
- **Services booking type**

## Technologies and Tools Used

React Native, Node.js, Firebase

## Our Services Reflected in This Project

This project drew on our Mobile Apps service, using React Native for cross-platform iOS/Android development, Node.js for the backend, and Firebase for infrastructure.
