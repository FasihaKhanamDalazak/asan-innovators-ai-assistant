# Asan Innovators — Case Study: Optex Opticians

Asan Innovators is a technology company headquartered in Nizampet, Hyderabad, Telangana, India. This is one of our 15 portfolio case studies.

## Project Identity

- **Client:** Optex Opticians (listed as "Eyewear")
- **Year:** 2024
- **Industry:** Healthcare

## Project Description

We built a website, mobile app, and Augmented Reality (AR) virtual try-on feature for Optex Opticians, an eyewear brand. The AR feature allows customers to virtually try on glasses before purchasing.

## Highlighted Results

- **AR virtual try-on feature**
- **Combined web and mobile app platform**
- **Eyewear brand type**

## Technologies and Tools Used

Next.js, React Native, AR (Augmented Reality), Stripe

## Our Services Reflected in This Project

This project drew on both our Web Development service (Next.js website) and Mobile Apps service (React Native app), combined with a custom Augmented Reality virtual try-on feature and Stripe payment integration.
